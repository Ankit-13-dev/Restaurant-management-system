# 🍽️ Restaurant Chain Management System

A MySQL database project for managing multiple branches of restaurant chains.

## 📌 Features
- Manage multiple restaurant brands and their branches
- Menu management with branch-specific pricing
- Order, payment, and inventory tracking
- Customer loyalty points system
- Stored procedures, triggers, and views

## 🗃️ How to Run
1. Open MySQL Workbench
2. Open `restaurant_management_system.sql`
3. Click Execute (⚡)
